== Description ==

Add iframes with shortcodes.


== Installation ==

1. Upload `iframe_shortcode.php` to the `wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!, there are no settings for this.
